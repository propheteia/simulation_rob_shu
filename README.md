Strive3d

Robocup 3d simulation league, Shanghai University, Strive Team.

1.How to install?

    mkdir build;
    cd build;
    cmake ..
    make
    make install

2.How does it work?
    
    Start simspark:

        $rcsoccersim3d 
    
    Excute start script:

        $sh start.sh localhost

    Kill agent process:
        
        $sh kill.sh
